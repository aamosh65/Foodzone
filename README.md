Food Zone
Food Zone is a React app that displays a list of foods and lets users filter them by category and search term in real time,
Using Vite+styled-components setup similar to common React starter projects.

Features

1.Search bar that filters foods by name as you type (case-insensitive).
​

2.Category pills (Allfoods, Lunch, Dinner, Breakfast, Dessert) to filter by category, with “Allfoods” showing every item.

3.Combined filtering: results match both the selected category and the search text.

4.Responsive card layout for each food item.
